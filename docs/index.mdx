# @sinoui/alert

警告提示，展现需要关注的信息。

[![npm version](https://img.shields.io/npm/v/@sinoui/alert)](https://www.npmjs.com/package/@sinoui/alert)
[![downloads](https://img.shields.io/npm/dm/@sinoui/alert)](https://www.npmjs.com/package/@sinoui/alert)

更多示例请参考[官方文档](https://sinoui.github.io/alert/)。

## 依赖安装

```shell
yarn add @sinoui/alert
```

## 入门示例

```tsx
import React from 'react';
import Alert from '@sinoui/alert';

function Demo() {
  return <Alert message="提示信息" />;
}
```

## API

Alert组件属性：

| 参数名称    | 参数类型        | 说明                                                         |
| ----------- | --------------- | ------------------------------------------------------------ |
| message     | ReactNode       | 警告提示内容                                                 |
| children    | React.ReactNode | 子元素，相当于 message   |
| type        | string          | 指定警告提示的样式，有四种选择 success、info、warning、error |
| description | ReactNode       | 警告提示的辅助性文字介绍                                     |
| showIcon    | boolean         | 是否显示辅助图标                                             |
| closable    | boolean         | 是否显示关闭按钮，默认为 false，即不展示                     |
| afterClose  | () => void      | 关闭动画结束后触发的回调函数                                 |
| className   | string          | 自定义类名                                                   |
